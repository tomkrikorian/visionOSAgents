# USD Command-Line Tools

Use this when you need a quick reference for common USD command-line utilities.

## Tool Summary

| Tool | Purpose |
|------|---------|
| `usdcat` | Convert between USD-compatible formats or output as text. |
| `usdchecker` | Validate USD files. |
| `usdrecord` | Render images of USD files. |
| `usdtree` | Output the hierarchy of USD files. |
| `usdzip` | Create USDZ files. |
| `usdcrush` | Compress the contents of USDZ files. |
| `usdextract` | Extract files from within USD-compatible packages. |

## Notes

- Use `usdchecker --arkit` for Apple platform validation.
- Run any tool with `--help` to see version-specific flags.
- See the `usd-editor` reference table for per-tool usage guides.
